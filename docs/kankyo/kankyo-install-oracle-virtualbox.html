Oracle VM VirtualBox のインストール¶

VirtualBox と Extension Pack をインストールします。

  1. VirtualBox-6.1.14-140239-Win.exe を実行


  2. Next をクリック


  3. Next をクリック


  4. Next をクリック


  5. Yes をクリック


  6. Install をクリック


  7. インストール中


  8. インストール(I) をクリック


  9. Start Oracle VM VirtualBox 6.1.14 after installation にチェックがついていることを確認


  10. 完了(F) をクリック


  11. ファイル(F) をクリック


  12. 環境設定(P)… をクリック


  13. 機能拡張 をクリック


  14. ＋ をクリック


  15. Oracle VM VirtualBox Extension Pack を選択


  16. 開く(O) をクリック


  17. インストール をクリック


  18. 「VirtualBox ライセンス」を下までスクロール　→　同意します(A) をクリック


  19. OK をクリック


  20. OK をクリック


  21. インストール終了
