VMware Workstation Pro のインストール¶

VMware Workstation 16 Pro （試用版）を使用しています。

インストール¶

  1. VMware-workstation-full-16.0.0-16894299.exe を実行


  2. 次へ(N) をクリック


  3. 使用許諾契約書に同意します(A) をチェック


  4. 次へ(N) をクリック


  5. 次へ(N) をクリック


  6. 自分の環境に合わせてチェック


  7. 次へ(N) をクリック


  8. 次へ(N) をクリック


  9. インストール(I) をクリック


  10. インストール中


  11. 完了(F) をクリック


ライセンス登録¶

  1. デスクトップ上のアイコンから起動


  2. VMware Workstation 16 の 30 日間使用を希望します(W) を選択


  3. 続行(C) をクリック


  4. 完了(F) をクリック


  5. VMware Workstation Pro が起動


  6. 右上の × をクリックして閉じる
