ヘルプ¶

?

? でコマンド全体のヘルプを、コマンドの後に ? を付けると、そのコマンドのヘルプを表示する。


IP アドレスの設定／確認¶

ip <ip address> <subnetmask> <default gateway>
ip dns <dns server>
ip dhcp
show ip

ホスト名の変更¶

set pcname <hostname>

疎通確認¶

ping <ip address>
trace <ip address>
show arp

設定の保存／読み込み¶

save <filename>
load <filename>
