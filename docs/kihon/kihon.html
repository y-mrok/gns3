起動¶

GNS3 を起動すると連動して GNS3 VM も起動します。

  1. GNS3 を起動


  2. GNS3 の起動終了　→　プロジェクトの指定画面を表示


  3. 遅れて GNS3 VM が自動的に起動　→　OK を押下


  4. GNS3 VM の起動終了


終了¶

GNS3 を終了すると GNS3 VM も終了します。

  1. File をクリック


  2. Quit をクリック


  3. 終了中（この画面の表示中に GNS3 VM が終了する）


  4. 終了


IOS router テンプレートの作成¶

  1. Edit をクリック


  2. Preferences… をクリック


  3. IOS routers をクリック


  4. New をクリック


  5. Run this IOS router on GNS3 VM を選択


  6. Next をクリック


  7. Browse… をクリック


  8. IOS ファイルを選択


  9. 開く(O) をクリック


  10. Yes をクリック


  11. IOS ファイルのロード中


  12. Next をクリック


  13. Next をクリック


  14. Next をクリック


  15. Next をクリック


  16. Idle-PC finder をクリック


  17. Idle-PC 値の計算中


  18. OK をクリック


  19. Finish をクリック


  20. OK をクリック


  21. IOS router テンプレートの作成終了


  22. ルーターのアイコンをクリック


  23. 取り込んだ IOS ファイルのルーターを確認


新規プロジェクトの作成¶

GNS3 は作成したネットワークなどは「プロジェクト」に保存します。そのため、ネットワークを作成する前にプロジェクトを作成しなければなりません。

  1. File をクリック


  2. New blank project をクリック


  3. Name にプロジェクト名を入力


  4. Location にプロジェクトを保存するフォルダーを指定


  5. OK をクリック


  6. プロジェクトの作成終了　→　左上にプロジェクト名を表示


  7. 作成したプロジェクトファイル


プロジェクトの保存¶

作成したネットワークをプロジェクトに保存します。

  1. File をクリック


  2. Save project as… をクリック


  3. Name と Location を確認


  4. OK をクリック


  5. プロジェクトの保存終了


プロジェクトのロード¶

保存済みのプロジェクトをロード（呼び出）します。

  1. File をクリック


  2. Open project をクリック


  3. ロードするプロジェクトファイルを指定


  4. 開く(O) をクリック


  5. プロジェクトのロード終了
