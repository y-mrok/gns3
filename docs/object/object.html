オブジェクトの配置¶

各オブジェクトはドラッグ＆ドロップで配置します。今回はルーターを配置・設定します。

  1. ルーターのアイコンをクリック


  2. ルーターをドラッグ＆ドロップで配置

    ↓


オブジェクトのカスタマイズ¶

配置したルーターのスロットにインターフェースを設定します。

  1. 配置したルーターを右クリック


  2. Configure をクリック


  3. Slots をクリック


  4. slots の初期状態（何も設定されていない）


  5. プルダウンリストから選択

    • NM-1FE-TX (FastEthernet × 1 ポート)

    • NM-1E (Ethernet × 1 ポート)

    • NM-4E (Ethernet × 4 ポート)

    • NM-16ESW (Ethernet 16 ポート)

    • NM-4T (Serial × 4 ポート)


  6. slot 0 と slot 1 に NM-1FE-TX を設定


  7. OK をクリック


  8. ルーターのカスタマイズ終了


オブジェクトの電源 ON¶

  1. 配置したルーターを右クリック


  2. Start をクリック


  3. ルーターの電源 ON（ Node が緑色に変更になった）


オブジェクトの電源 OFF¶

  1. 配置したルーターを右クリック


  2. Stop をクリック


  3. ルーターの電源 OFF（ Node が赤色に変更になった）


すべてのオブジェクトの電源 ON¶

  1. Start/Resume all nodes をクリック


  2. すべてのオブジェクトの電源 ON 終了


すべてのオブジェクトの電源 OFF¶

  1. Stop all nodes をクリック


  2. :guilabel:`` をクリック


コンソールを起動¶

パソコンとルーターをコンソールケーブルで接続し、ターミナルソフト（今回は TeraTerm ）を起動した場合と同等のことが行えます。

  1. 配置したルーターを右クリック


  2. Console をクリック


  3. Tera Term が起動


Note

Tera Term を閉じるとコンソールは切断されます

オブジェクトをケーブルで接続¶

  1. ケーブルのアイコンをクリック


  2. オブジェクトをクリック


  3. ケーブルを接続するインターフェースをクリック


  4. 接続先のオブジェクトをクリック


  5. ケーブルを接続するインターフェースをクリック


  6. ケーブルのアイコンをクリック


  7. オブジェクト間のケーブル接続終了


インターフェース名の表示／非表示¶

各オブジェクトのインターフェース名を表示／非表示します。

表示¶

  1. View をクリック


  2. Show/Hide interface label をクリック


  3. インターフェース名を表示


非表示（消去）¶

  1. View をクリック


  2. Show/Hide interface label をクリック


  3. インターフェース名を非表示（消去）


パケットキャプチャ¶

Wireshark を使用してオブジェクト間のパケットをキャプチャできます。今回は R1 と R2 間のパケットキャプチャを例に説明します

キャプチャ開始¶

  1. マウスカーソルを R1 と R2 を結ぶケーブルにあわせる（ケーブルが赤くなります）


  2. 右クリック　→　メニューを表示


  3. Start capture をクリック


  4. OK をクリック


  5. Wireshark が起動してキャプチャを開始


  6. キャプチャー中は対象の LAN ケーブル上に虫眼鏡のアイコンが表示される


キャプチャ停止¶

  1. マウスカーソルを R1 と R2 を結ぶケーブルにあわせる（ケーブルが赤くなります）


  2. 右クリック　→　メニューを表示


  3. Stop capture をクリック


  4. キャプチャ停止（ケーブル上の虫眼鏡が消える）



Caution

Stop capture でキャプチャを停止しても Wireshark は起動したままです。
