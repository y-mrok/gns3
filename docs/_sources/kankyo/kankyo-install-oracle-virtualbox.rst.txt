**************************************************
Oracle VM VirtualBox のインストール
**************************************************
VirtualBox と Extension Pack をインストールします。

#. :file:`VirtualBox-6.1.14-140239-Win.exe` を実行

   .. image:: img/2020-09-06_18h35_27.png

   |

#. :guilabel:`Next` をクリック

   .. image:: img/2020-09-06_18h35_57.png

   |

#. :guilabel:`Next` をクリック

   .. image:: img/2020-09-06_18h36_18.png

   |

#. :guilabel:`Next` をクリック

   .. image:: img/2020-09-06_18h36_40.png

   |

#. :guilabel:`Yes` をクリック

   .. image:: img/2020-09-06_18h36_55.png

   |

#. :guilabel:`Install` をクリック

   .. image:: img/2020-09-06_18h37_32.png

   |

#. インストール中

   .. image:: img/2020-09-06_18h37_53.png

   |

#. :guilabel:`インストール(I)` をクリック

   .. image:: img/2020-09-06_18h38_55.png

   |

#. :guilabel:`Start Oracle VM VirtualBox 6.1.14 after installation` にチェックがついていることを確認

   .. image:: img/2020-09-06_18h39_17.png

   |

#. :guilabel:`完了(F)` をクリック

   .. image:: img/2020-09-06_18h43_01.png

   |

#. :guilabel:`ファイル(F)` をクリック

   .. image:: img/2020-09-06_18h45_04.png

   |

#. :guilabel:`環境設定(P)...` をクリック

   .. image:: img/2020-09-06_18h45_31.png

   |

#. :guilabel:`機能拡張` をクリック

   .. image:: img/2020-09-06_18h46_03.png

   |

#. :guilabel:`＋` をクリック

   .. image:: img/2020-09-06_18h46_20.png

   |

#. Oracle VM VirtualBox Extension Pack を選択

   .. image:: img/2020-09-06_18h47_16.png

   |

#. :guilabel:`開く(O)` をクリック

   .. image:: img/2020-09-06_18h47_43.png

   |

#. :guilabel:`インストール` をクリック

   .. image:: img/2020-09-06_18h48_06.png

   |

#. 「VirtualBox ライセンス」を下までスクロール　→　:guilabel:`同意します(A)` をクリック

   .. image:: img/2020-09-06_18h48_50.png

   |

#. :guilabel:`OK` をクリック

   .. image:: img/2020-09-06_18h49_16.png

   |

#. :guilabel:`OK` をクリック

   .. image:: img/2020-09-06_18h49_33.png

   |

#. インストール終了

   .. image:: img/2020-09-06_18h50_00.png

   |
