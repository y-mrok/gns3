**************************************************
VMware Workstation Pro のインストール
**************************************************
VMware Workstation 16 Pro （試用版）を使用しています。

.. _kankyo-install-vmware-workstation-pro-install:

インストール
==================================================
#. :file:`VMware-workstation-full-16.0.0-16894299.exe` を実行

   .. image:: img/2020-09-19_11h12_39.png

   |

#. :guilabel:`次へ(N)` をクリック

   .. image:: img/2020-09-19_11h13_28.png

   |

#. :guilabel:`使用許諾契約書に同意します(A)` をチェック

   .. image:: img/2020-09-19_11h13_50.png

   |

#. :guilabel:`次へ(N)` をクリック

   .. image:: img/2020-09-19_11h14_02.png

   |

#. :guilabel:`次へ(N)` をクリック

   .. image:: img/2020-09-19_11h14_19.png

   |

#. 自分の環境に合わせてチェック

   .. image:: img/2020-09-19_11h15_03.png

   |

#. :guilabel:`次へ(N)` をクリック

   .. image:: img/2020-09-19_11h16_32.png

   |

#. :guilabel:`次へ(N)` をクリック

   .. image:: img/2020-09-19_11h16_47.png

   |

#. :guilabel:`インストール(I)` をクリック

   .. image:: img/2020-09-19_11h17_01.png

   |

#. インストール中

   .. image:: img/2020-09-19_11h17_23.png

   |

#. :guilabel:`完了(F)` をクリック

   .. image:: img/2020-09-19_11h19_27.png

   |


.. _kankyo-install-vmware-workstation-pro-license:

ライセンス登録
==================================================
#. デスクトップ上のアイコンから起動

   .. image:: img/2020-09-19_11h19_50.png

   |

#. :guilabel:`VMware Workstation 16 の 30 日間使用を希望します(W)` を選択

   .. image:: img/2020-09-19_11h21_03.png

   |

#. :guilabel:`続行(C)` をクリック

   .. image:: img/2020-09-19_11h21_16.png

   |

#. :guilabel:`完了(F)` をクリック

   .. image:: img/2020-09-19_11h21_36.png

   |

#. VMware Workstation Pro が起動

   .. image:: img/2020-09-19_11h22_16.png

   |

#. 右上の :guilabel:`×` をクリックして閉じる

   .. image:: img/2020-09-19_11h22_36.png

   |
