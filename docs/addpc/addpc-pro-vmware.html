追加する仮想マシンの準備¶

GNS3 に追加する仮想マシンは他の VMware Workstation Pro 上の仮想マシンと同様にインストールします。ツールなどが必要な場合は、それらもインストールします。

今回追加する仮想マシンは Microsoft Windows 10 Enterprise Evaluation です。


準備¶

追加する仮想マシンのネットワークインターフェースは VMnet2 ～ VMnet19 （ VMnet8 は除く）のいずれかに紐づける必要があります。 VMnet2 ～ VMnet19 の設定は「仮想ネットワークエディタ」で行います。

VMnet2 ～ VMnet19 の要件¶

  • Hos-Only であること

  • DHCP が稼働していないこと

設定¶

今回は VMnet2 を設定します。

仮想ネットワークエディタ¶

  1. 「仮想ネットワークエディタ」を起動


  2. 設定の変更(C) をクリック


  3. ネットワークの追加(E)… をクリック


  4. 追加するネットワークを選択(S) は VMnet2 を選択


  5. OK をクリック


  6. VMnet2 を選択　→　次のように設定

    項目

    設定内容

    VMnet 情報

    ホストオンリー（ VM をプライベートネットワーク内で内部的に接続する）(H)

    ローカル DHCP サービスを使用して IP アドレスを VM に配布する(D)

    チェックを外す

  7. OK をクリック


  8. 設定終了


GNS3¶

  1. GNS3 を起動


  2. Edit をクリック


  3. Preferences… をクリック


  4. VMware をクリック


  5. Advanced local settings をクリック


  6. vmnet19 を vmnet2 に変更

    ↓


  7. Configure をクリック


  8. 設定中


  9. OK をクリック


  10. 設定終了


仮想マシンの追加¶

準備した仮想マシン を GNS3 に追加します。

VMware Workstation Pro¶

VMware Workstation Pro で仮想マシンのネットワークインターフェースを VMnet2 ～ VMnet19 （ VMnet8 は除く）のいずれかに紐づけます。今回は先に設定した VMnet2 に紐づけました。



GNS3¶

  1. Edit をクリック


  2. Preferences… をクリック


  3. VMware VMs をクリック


  4. New をクリック


  5. Next をクリック


  6. VM list のプルダウンリストから追加する仮想マシンを選択

    ↓


  7. Finish をクリック


  8. Edit をクリック


  9. Network をクリック


  10. Allow GNS3 to override non custom VMware adapter をチェック


  11. OK をクリック


  12. OK をクリック


  13. Browse End Devices をクリック


  14. 追加した仮想マシンを確認


追加した仮想マシンを使う¶

追加した仮想マシンは他のオブジェクトと同様にドラッグ＆ドロップでネットワーク図に追加できます。 ウォークスルー で作成したネットワーク図から PC1 を削除し、追加した仮想マシン（ Windows 10 x64 ）を配置しました。

すべてのオブジェクトの電源を ON にしました。

追加した仮想マシンの電源が ON になると自動的に起動します（起動に時間を要します）。

仮想マシンの IP アドレスは手動で設定します。

仮想マシンから PC2 へ ping コマンドの実行結果です。疎通ができています。

ネットワーク図上の仮想マシンの電源を OFF にすると、仮想マシンは自動でシャットダウンします。
