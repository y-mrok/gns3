コンソールを起動
==================================================
パソコンとルーターをコンソールケーブルで接続し、ターミナルソフト（今回は TeraTerm ）を起動した場合と同等のことが行えます。

#. 配置したルーターを右クリック

   .. image:: img/2020-09-05_14h35_36.png

   |

#. :guilabel:`Console` をクリック

   .. image:: img/2020-09-05_14h35_51.png

   |

#. Tera Term が起動

   .. image:: img/2020-09-05_14h37_16.png

   |

.. note::

   Tera Term を閉じるとコンソールは切断されます
